# AndroidBridge 1.1.0

AndroidBridge 1.1.0 makes transfer workflows faster, safer, and easier to control.

## Highlights

- Upload multiple Mac files or folders in one action.
- Select and download multiple Android files or folders at once.
- Choose the Mac folder for downloads instead of always saving to Downloads.
- Cancel active transfers from the toolbar or status bar.
- View upload progress, download progress, elapsed time, and ETA where available.
- See clearer ADB error messages for unauthorized, offline, missing, or permission-denied devices and files.

## Validation

- `swift run AndroidBridgeCoreTests`
- `swift build`
- `./script/build_and_run.sh --verify`
- `./script/package_dmg.sh`

## Install

```bash
brew tap togiwan/tap
brew install --cask androidbridge
```

Manual install: download `AndroidBridge.dmg`, open it, and copy `AndroidBridge.app` to Applications.
