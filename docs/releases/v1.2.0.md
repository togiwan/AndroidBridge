# AndroidBridge 1.2.0

## Highlights

- Adds ADB Wireless pairing, discovery, connect, and disconnect controls.
- Reuses the existing file browser for wireless debugging devices, so USB and Wi-Fi transfers share the same workflow.
- Discovers Android wireless debugging pairing and connection services on the local network.
- Keeps multi-select upload/download, folder transfer, preview, cancel, progress, and ETA support available over Wi-Fi.

## Verification

- `swift run AndroidBridgeCoreTests`
- `swift build`
- `./script/build_and_run.sh --verify`
- `./script/package_dmg.sh`

## Install

```bash
brew tap togiwan/tap
brew install --cask androidbridge
```

Manual install: download `AndroidBridge.dmg`, open it, and copy `AndroidBridge.app` to Applications.
